# Sortini – Sorting Algorithm Visualizer

**Program Type:** GUI-based (Web Application)

Sortini is a graphical, browser-based program that allows users to upload a text file containing numbers, choose a sorting algorithm, select sorting order, and view the sorted results along with execution time.

## Features
- Graphical User Interface (GUI) using HTML, CSS, and JavaScript
- Supports Bubble Sort, Insertion Sort, and Merge Sort
- Ascending and Descending sorting options
- File upload support for `.txt` datasets
- Displays sorted output and time taken

## Requirements
- Any modern web browser (Chrome, Edge, Firefox, etc.)
- No additional installations required

## Files
- `index.html` – main GUI application
- `dataset.txt` – sample input file (numbers listed one per line)
- `sunshet.jpg` – background image (optional, for design)

## How to Run
1. Make sure all files are in the same folder.
2. Open `index.html` in a web browser.
3. Upload a `.txt` file containing integers.
4. Select a sorting algorithm and sorting order.
5. Click **Ascending** or **Descending** to run the sort.
